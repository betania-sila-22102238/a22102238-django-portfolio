Front-end

O front-end é a parte visual de um site ou aplicação web. É responsável por criar a interface que o usuário irá interagir. As tecnologias utilizadas no front-end são:

  • HTML
  • CSS
  • JavaScript

O HTML é responsável por criar a estrutura do site, indicando o conteúdo que deve ser exibido. O CSS é responsável por definir o estilo visual do site, como cores, fontes e layout. Já o JavaScript é responsável por adicionar interatividade ao site, permitindo a criação de animações, validação de formulários, entre outras funcionalidades.
