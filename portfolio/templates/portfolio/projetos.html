Estes projetos foram realizados durante o meu percurso acadêmico na universidade, onde pude aplicar e aprimorar meus conhecimentos com muito gosto e dedicação. Aqui estão alguns dos meus trabalhos mais notáveis:

Minesweeper

Minesweeper é um jogo de quebra-cabeça baseado em lógica. O objetivo é limpar o campo minado sem detonar nenhuma mina. Os jogadores devem usar as dicas numéricas para evitar as minas escondidas e marcar suas localizações com bandeiras. Estratégia e lógica são essenciais para completar o tabuleiro com sucesso.

Não te Constipes

Não Te Constipes é um jogo de tabuleiro para dois jogadores, cada um com uma família de 4 peões. O objetivo é completar uma volta ao tabuleiro com todos os peões. Se um peão for ultrapassado por outro da família oponente, ele fica constipado e volta para a casa mãe. Ganha o primeiro jogador a completar a volta com todos os peões.

Deisiflix

Deisiflix é uma aplicação em Java para análise de estatísticas de filmes. Permite responder a consultas sobre dados de filmes, realizar cálculos estatísticos, gerar relatórios e oferecer uma interface amigável. Foi desenvolvida para aplicar conhecimentos em Java e análise de dados, explorando o mundo do cinema por meio de estatísticas.

Deisi Jungle

Deisi Jungle é um jogo onde os animais competem em uma série de competições esportivas para determinar o atleta supremo da selva. O jogador assume o papel de um dos animais e compete em corridas e outros desafios atléticos. O objetivo é vencer a maioria das provas e ser coroado como o Rei da Selva.

Máquina de Lavar Roupa

Este projeto simula a automação de uma máquina de lavar roupa, controlando e otimizando todas as etapas do ciclo de lavagem. Inclui uma interface para selecionar programas, ajustar a temperatura da água, definir o nível de agitação e agendar inícios de lavagem. O sistema toma decisões inteligentes para eficiência energética e conveniência.

Aplicação de Mobilidade Urbana

A Aplicação de Mobilidade Urbana visa auxiliar os condutores na busca por estacionamentos disponíveis na cidade de Lisboa, gerenciados pela EMEL. O projeto envolve a implementação de uma interface intuitiva para relatar incidentes, verificar disponibilidade de vagas e obter direções para o estacionamento desejado. O objetivo é melhorar a mobilidade urbana e facilitar a vida dos condutores na cidade. Baixar APK

EMEL - Gestão de Estacionamentos

A aplicação EMEL tem como objetivo facilitar a gestão e o acesso aos parques de estacionamento em Lisboa. Inclui funcionalidades como localização de parques disponíveis, estado de ocupação em tempo real, registo de incidentes e navegação até os parques. O projeto foi desenvolvido com foco na usabilidade e experiência do utilizador, utilizando tecnologias como Axure para prototipagem e testes de usabilidade.

Projecto Concessionário Praias Algarve

O Projecto Concessionário Praias Algarve foi desenvolvido para gerenciar reservas de sombrinhas e espreguiçadeiras nas praias A, B e C. Utilizando tecnologias como Apache Tomcat, Webservices REST e SOAP, e comunicação RMI para o backend, o sistema permite reservar, cancelar e listar sombrinhas disponíveis. A autenticação e o registo de utilizadores são realizados no frontend, enquanto os dados de reservas são mantidos no backend.

Projecto SeguidorDeLinha-EV3

Este trabalho teve por base os conhecimentos adquiridos na disciplina de Arquiteturas Avançadas de Computadores no referente a Arduino Uno R3 compreendendo a parte de hardware e software. Construir um projeto de carro Arduino pode ser uma tarefa desafiadora, exigindo conhecimentos de programação, eletrônica e robótica. No entanto, com as habilidades certas, foi uma experiência muito gratificante e divertida.

Uber de Transporte de Mercadorias

O projeto "Uber de Transporte de Mercadorias" foi desenvolvido como parte do Trabalho Final de Curso (TFC) na Licenciatura em Engenharia Informática. Este projeto visa criar uma plataforma centralizada para facilitar o transporte eficiente e seguro de mercadorias. Inspirado no modelo de negócios da Uber, mas focado na logística de mercadorias.
